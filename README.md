# Magic Hands - موقع إيكومرس جزائري

## نظرة عامة
Magic Hands هو موقع تجارة إلكترونية جزائري متخصص في بيع الحقائب والإكسسوارات مع إمكانية التخصيص والرسم عليها.

## المميزات

### الموقع الرئيسي (index.html)
- ✅ واجهة عربية احترافية متجاوبة مع جميع الأجهزة
- ✅ دعم 3 لغات: العربية، الفرنسية، والإنجليزية
- ✅ عرض المنتجات (حقائب وإكسسوارات)
- ✅ إمكانية اختيار اللون والكمية
- ✅ خيار إضافة رسمة مخصصة على المنتج
- ✅ سلة تسوق ديناميكية
- ✅ نظام الطلبات مع اختيار الولاية
- ✅ حساب تلقائي لرسوم التوصيل حسب الولاية

### لوحة التحكم (admin.html)
- ✅ لوحة معلومات شاملة (Dashboard)
- ✅ إدارة المنتجات (إضافة، تعديل، حذف)
- ✅ رفع صور المنتجات
- ✅ إدارة الطلبات وتتبع حالتها
- ✅ إعدادات التوصيل (اسم مكتب التوصيل)
- ✅ تحديد أسعار التوصيل لكل الولايات (58 ولاية)
- ✅ إحصائيات شاملة

## كيفية الاستخدام

### 1. فتح الموقع
- افتح `index.html` لعرض الموقع الرئيسي للعملاء
- افتح `admin.html` لإدارة الموقع

### 2. إضافة المنتجات (من لوحة التحكم)
1. افتح `admin.html`
2. اذهب إلى قسم "المنتجات"
3. اضغط "إضافة منتج جديد"
4. أدخل المعلومات بالثلاث لغات
5. أضف الألوان المتاحة (مفصولة بفاصلة)
6. ارفع صور المنتج
7. احفظ المنتج

### 3. إدارة أسعار التوصيل
1. من لوحة التحكم، اذهب إلى "التوصيل والأسعار"
2. أدخل اسم مكتب التوصيل
3. حدد سعر التوصيل لكل ولاية
4. احفظ التغييرات

### 4. إدارة الطلبات
1. اذهب إلى قسم "الطلبيات"
2. اضغط على أي طلب لعرض التفاصيل
3. قم بتحديث حالة الطلب (قيد الانتظار، مؤكدة، تم الشحن، تم التوصيل، ملغاة)

## التقنيات المستخدمة
- **HTML5** - الهيكل الأساسي
- **CSS3** - التصميم والتنسيق المتجاوب
- **JavaScript (ES6)** - الوظائف والتفاعل
- **LocalStorage** - تخزين البيانات محلياً

## المميزات التقنية
- ✅ تصميم متجاوب (Responsive) يعمل على جميع الأجهزة
- ✅ دعم RTL للعربية و LTR للغات الأخرى
- ✅ نظام ترجمة ديناميكي
- ✅ حفظ البيانات محلياً (لا حاجة لقاعدة بيانات)
- ✅ واجهة سهلة الاستخدام
- ✅ تصميم احترافي وعصري

## للتطوير المستقبلي
لجعل الموقع يعمل أونلاين بشكل كامل، يمكنك:

1. **استخدام Backend**:
   - Node.js + Express
   - PHP + MySQL
   - Firebase (سهل للمبتدئين)

2. **رفع الموقع على استضافة**:
   - Netlify (مجاني)
   - Vercel (مجاني)
   - GitHub Pages (مجاني)
   - استضافة مدفوعة جزائرية

3. **إضافة نظام دفع**:

4. **إضافة قاعدة بيانات**:
